diet

a schema for representing dietary preferences

The diet schema provides RDFa properties to mark a person's dietary preference.

This is useful as it allows applications to aggregate said preferences, allowing for booking of restaurants and catering to provide for vegetarians, vegans, people with dietary preferences etc.

dietary preference

cardinality
0 to infinity
description

Relates a person to their dietary preference. A person can have zero or more dietary preferences: a dietary preference includes allergies, sensitivies, religious restrictions and any other thing which is useful to know when preparing food for a person.

Allergies, intolerances etc.
